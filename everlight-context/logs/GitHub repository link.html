Womack Estate Case Portal

A comprehensive database-powered web portal for analyzing the Womack Estate investigation timeline spanning 1989-2025, featuring 40+ documented events, evidence tracking, and advanced filtering capabilities.

Features

  • Complete Timeline: 35-year chronological case analysis with 40+ events
  • Advanced Filtering: Search by priority, year, tags, and full-text search
  • Evidence Tracking: 29+ documented evidence items with status tracking
  • Financial Analysis: $470,000+ in tracked transactions and suspicious activity
  • People Database: Key individuals with aliases and relationship mapping
  • Priority Classification: Critical, Important, and Normal event categorization
  • Real-time Dashboard: Live statistics and case metrics

Database Schema

The application uses PostgreSQL with the following main tables: - timeline_events - Chronological case events with priority and financial impact - people - Individuals involved with aliases and roles - evidence - Document and evidence inventory with discovery status - financial_transactions - Financial activity tracking - properties - Real estate transactions and ownership changes - legal_documents - Legal filings and court documents

Quick Start

Option 1: Replit Deployment

  1. Fork this Replit project
  2. The PostgreSQL database is pre-configured
  3. Click "Run" to start the application
  4. Access the portal at the provided URL

Option 2: Self-Deployment

Prerequisites

  • Python 3.11+
  • PostgreSQL 12+
  • Git

@app.route('/api/timeline')
def get_timeline():
    """Get timeline events from database"""
    try:
        conn = get_db_connection()
        cur = conn.cursor(cursor_factory=psycopg2.extras.RealDictCursor)
        cur.execute("""
            SELECT id, date, title, people_involved, summary, tags, evidence_links, 
                   priority, financial_impact, legal_significance
            FROM timeline_events 
            ORDER BY date
        """)
        events = cur.fetchall()
        cur.close()
        conn.close()

        return jsonify([{
            'ID': event['id'],
            'Date': event['date'].isoformat() if event['date'] else None,
            'Title': event['title'],
            'People Involved': event['people_involved'],
            'Summary': event['summary'],
            'Tags': event['tags'],
            'Evidence Links': event['evidence_links'],
            'Priority': event['priority'],
            'Financial Impact': float(event['financial_impact']) if event['financial_impact'] else None,
            'Legal Significance': event['legal_significance']
        } for event in events])
    except Exception as e:
        return jsonify({'error': str(e)}), 500

def parse_date(date_str):
    """Parse various date formats"""
    if not date_str or date_str.strip() == '':
        return None

    date_str = date_str.strip()

    # Try different date formats
    formats = [
        '%Y-%m-%d',
        '%m/%d/%Y',
        '%Y-%m-%d %H:%M:%S',
        '%Y',  # Just year
        '%B %d, %Y',  # Month Day, Year
    ]

    for fmt in formats:
        try:
            if fmt == '%Y' and len(date_str) == 4:
                return datetime.strptime(f"{date_str}-01-01", '%Y-%m-%d').date()
            return datetime.strptime(date_str, fmt).date()
        except ValueError:
            continue

    return None
